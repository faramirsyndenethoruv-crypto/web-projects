## 📁 Project Structure

```

│   └── business-card-marie/   # With complete assets
├── shared/                    # Reusable components
│   ├── components/           # HTML/JS components  
│   ├── styles/              # CSS frameworks
│   └── images/              # Common assets
├── scripts/                  # Build & deploy tools
│   ├── web-generator.js     # Main generator script
│   ├── image-generator.js   # AI image creation
│   └── deploy.js           # Deployment automation
└── .github/workflows/       # CI/CD pipelines
```
